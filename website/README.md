# Website

Static landing page for Valor.

## Files

- `index.html`
- `styles.css`

## Local Preview

From the repo root:

```bash
python3 -m http.server 8000 --directory website
```

Then open:

- `http://127.0.0.1:8000/`

## Deployment

This folder is designed to deploy as a static site to a subdomain on a personal
domain without requiring a backend.
