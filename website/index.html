What Valor is

A local core for capturing the work that commits miss.

Debugging, design trade-offs, mentoring, operational ownership, cross-team alignment, and investigative work rarely show up clearly in merge history.

Valor is designed around a different assumption: developers already spend a meaningful part of their day working through assistants and agents. That interaction layer can become a useful, inspectable coaching loop instead of a black box.

What ships today

A focused starting point, not a platform sprawl.

  • Morning briefing, weekly reflection, evening wrap-up
  • PR review coach, design doc coach, task identifier
  • Local evidence CLI and local storage model
  • Bring-your-own career framework
  • Inspectable prompts, rules, and Python logic

Trust boundary

Local-first, with explicit limits.

This repo does

  • Store local core data on your machine
  • Keep the core inspectable
  • Avoid built-in telemetry and cloud sync

This repo does not

  • Promise an air-gapped workflow by itself
  • Override your host assistant’s model or network behavior
  • Act as an employer-facing dashboard
