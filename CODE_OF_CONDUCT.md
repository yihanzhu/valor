# Code of Conduct

Valor is a collaboration space for thoughtful, privacy-conscious work. We want
contributors and users to feel safe asking questions, sharing ideas, and giving
direct feedback without being belittled.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being honest about disagreements.
- Keep feedback specific and actionable.
- Be mindful that this project touches sensitive career and workplace concerns.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Doxxing or sharing private information without consent
- Dismissive or demeaning language aimed at newcomers
- Deliberately unsafe suggestions around confidential or sensitive data

## Scope

This applies to repository discussions, issues, pull requests, and other public
project spaces.

## Enforcement

Project maintainers may edit, remove, or moderate contributions that conflict
with this code of conduct.
